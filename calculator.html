📖 How to Use

🔢 Basic Operations

  • Click number buttons to enter values
  • Use +, −, ×, ÷ for addition, subtraction, multiplication, division
  • Press = to see the result
  • Press AC to clear and start over
    • Example: 15 + 8 = 23

📐 Trigonometric Functions

  • sin, cos, tan - Calculate sine, cosine, tangent (angles in degrees)
  • Enter the angle value first, then click the function button
    • Example: sin(90°) = 1
    Example: cos(0°) = 1

🔬 Advanced Math Functions

  • √ - Square root of a number
  • x² - Square a number (multiply by itself)
  • 1/x - Calculate reciprocal (1 divided by the number)
  • % - Modulo (find remainder)
    • Example: √16 = 4
    Example: 5² = 25

📊 Logarithmic Functions

  • log - Base 10 logarithm
  • ln - Natural logarithm (base e)
  • e^x - e raised to the power x
    • Example: log(100) = 2
    Example: ln(1) = 0

✨ Special Functions

  • π - Display the value of pi (3.14159...)
  • n! - Calculate factorial (e.g., 5! = 5×4×3×2×1 = 120)
  • +/− - Toggle sign (convert positive to negative and vice versa)
    • Example: 5! = 120
    Example: −(25) shows -25

💡 Tips & Tricks

  • You can use the decimal point (.) for non-whole numbers
  • Chain operations together: 5 + 3 × 2 = calculate for complex math
  • Use scientific functions for engineering and physics problems
  • Press the same operation twice to perform it instantly
    • Example: 5 + 3 × 2 = 16
